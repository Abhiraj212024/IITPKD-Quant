# Quant Hobby Group - Educational Repository

Welcome to the Quant Hobby Group repository! This is a collection of lesson codes and educational content from lectures conducted by the Quant Hobby Group. Our focus is on quantitative finance, data collection, and stochastic modeling for financial analysis.

## 📚 About

This repository contains:
- **Lesson Materials**: Interactive Jupyter notebooks covering various quantitative finance topics
- **Reusable Library**: Modular Python code for data collection and advanced filtering techniques
- **Practical Examples**: Real-world applications of concepts discussed in lectures


## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- pip or conda

### Installation

1. Clone this repository:
```bash
git clone <repository-url>
cd Quant_Club
```

2. Create a virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```
